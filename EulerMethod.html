Euler's Method

This program offers two different implementations of Euler's Method for solving differential equations when the differential equation is in the form y' = f(x,y) and an initial value is given. Euler's Method can be used to find the value of the function y at a specified point x using information about the slope of y stored in the differential equation. The implementing methods in this program solve Euler's equations and return an array of x and y values for each step along the interval. The last entries in the array are the specified point x and the approximate value of the function y at that point. In this implementation, the interval parameter specifies the final x value for which a estimate of y is required.
